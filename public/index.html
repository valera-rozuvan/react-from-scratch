Looking at a GPG key is like looking at a fractal.

There is hidden beauty everywhere.

But you can’t quite describe what you are looking at 😉

This is a minimal React application which provides the functionality to generate a GPG key online.

The awesome library OpenPGP.js is used to work with GPG keys in the browser.

No back-end calls are made - purely in-browser experience!

Find the full source code of the application at https://github.com/valera-rozuvan/react-from-scratch
